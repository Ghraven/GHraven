### 🤖 What I build

I build AI-powered tools and multi-agent systems — apps where multiple LLMs collaborate, debate, and reason together in real time. I focus on making AI practical: local models via Ollama, cloud API integrations, and real web apps people can actually use.

---

### 🚀 Featured Project

**[🏠 Agent Meeting Room](https://github.com/Ghraven/agent-meeting-room)** &nbsp;·&nbsp; `Python` `Flask` `Ollama` `Claude API`

> A web app where you @mention AI agents to bring them into a live group chat.
> 4 local models (Mistral, Phi3, Gemma2, DeepSeek) + Claude on demand.
> Includes structured **debate mode** (3 rounds), **free-talk streaming** via SSE, and **Obsidian memory** integration.
